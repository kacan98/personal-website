# AI Job Application Platform

A full-stack platform for tailoring CVs, generating cover letters, and working across multiple languages.

## Scope

- CV customization flows
- Cover-letter generation
- Multi-language support
- Workflow support for repeated job applications

## Focus

The interesting part of the project is not AI as a buzzword. It is building a practical system around real application workflows.

# Mobilförst sportlagsorganisatör

En samutvecklad app för att automatisera lagindelning, poängräkning och rapportering för padel och biljard, med fokus på att fungera bra direkt i mobilen.

## Funktioner

- **Automatisk lagindelning** - En smart logik för att skapa jämna lag
- **Poängspårning** - Resultat uppdateras löpande
- **Matchrapportering** - Statistik och historik samlas på ett ställe
- **Mobilförst design** - Anpassad för telefoner och surfplattor
- **Realtidsuppdateringar** - Synk mellan enheter utan extra manuellt arbete

## Teknisk stack

- **Frontend:** Angular, TypeScript
- **Styling:** Modern CSS med responsiv design
- **Deployment:** GitHub Pages
- **Arkitektur:** Komponentbaserad med reaktiva mönster

## Vad jag lärde mig

Det här projektet lärde mig mycket om mobile-first-design och om att bygga tillsammans med någon annan. Att arbeta i ett delat kodflöde gav mig mer praktisk förståelse för kodgranskning, ansvarsfördelning och git-arbetsflöden i ett team.

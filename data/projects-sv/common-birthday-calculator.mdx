# Kalkylator för gemensam födelsedag

Beräknar hur sannolikt det är att personer i en grupp delar födelsedag.

## Funktioner

- **Analys för grupper** - Räknar ut hur vanliga gemensamma födelsedagar är i olika gruppstorlekar
- **Sannolikhetsberäkningar** - Visar den matematiska sannolikheten bakom sammanträffandena
- **Interaktivt gränssnitt** - En enkel vy där man snabbt kan testa olika scenarier
- **Tydliga resultat** - Presenterar utfallet på ett sätt som är lätt att förstå

## Teknisk stack

- **Frontend:** Angular, TypeScript
- **Styling:** Modern CSS med responsiv design
- **Deployment:** GitHub Pages
- **Matematik:** Statistiska sannolikhetsberäkningar

## Vad jag lärde mig

Det här projektet lärde mig hur man gör abstrakt sannolikhetslära begriplig i ett webbgränssnitt. Det gav mig också mer erfarenhet av komponentbaserad utveckling i Angular och hur TypeScript hjälper när logiken snabbt blir mer matematisk.

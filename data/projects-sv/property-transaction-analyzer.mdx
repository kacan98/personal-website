# Fastighetsanalys - köpa eller hyra

Ett analysverktyg för att jämföra de ekonomiska konsekvenserna av att köpa eller hyra bostad, byggt med React och Redux för att kunna visa detaljerade beräkningar och visualiseringar.

## Funktioner

- **Finansiell analys** - Jämför kostnader för köp och hyra över tid
- **Interaktiva diagram** - Gör utvecklingen lättare att förstå visuellt
- **Justerbara parametrar** - Ändra räntor, bostadspriser och tidshorisont
- **Detaljerade rapporter** - Bryter ned kostnader, antaganden och möjliga besparingar

## Teknisk stack

- **Frontend:** React, Redux, TypeScript
- **Styling:** Modern CSS med responsiv design
- **Deployment:** GitHub Pages
- **Tillståndshantering:** Redux Toolkit för komplexa finansiella beräkningar

## Vad jag lärde mig

Projektet fördjupade min förståelse för komplex state management i Redux och för hur man gör finansiella beräkningar begripliga i ett gränssnitt. Den stora utmaningen var inte bara matematiken, utan att presentera resultatet på ett sätt som fortfarande känns tydligt när siffrorna ändras.

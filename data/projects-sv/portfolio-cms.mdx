# Portfolio med CMS

En dynamisk portfolio kopplad till ett CMS så att även personer utan utvecklarbakgrund enkelt kan uppdatera innehållet.

## Funktioner

- **CMS-gränssnitt** - Enkel vy för att uppdatera innehåll
- **Dynamiskt innehåll** - Förändringar kan göras utan att röra koden
- **Responsiv design** - Fungerar bra på olika skärmstorlekar
- **SEO-stöd** - Grundläggande hantering av metadata och indexering
- **Prestandafokus** - Snabba laddtider och optimerade tillgångar

## Teknisk stack

- **Framework:** Next.js med TypeScript
- **Frontend:** React med moderna hooks
- **Styling:** CSS Modules med responsiv design
- **Innehållshantering:** Integrerad CMS-lösning
- **Deployment:** Vercel med automatiska deployments

## Vad jag lärde mig

Det här projektet lärde mig hur viktigt det är att bygga verktyg som fungerar för människor som inte tänker i kod. Att sätta upp ett CMS från grunden gav mig bättre förståelse för innehållsflöden och balansen mellan flexibilitet och enkelhet.

# App för butiksbetyg

En enkel webbapplikation där användare kan betygsätta och recensera lokala matbutiker. Jag byggde den som ett lärprojekt för att öva fullstackutveckling i Microsoft-miljö.

## Vad den gör

- **Sätt betyg på butiker** - Användare kan lämna stjärnbetyg
- **Skriv recensioner** - Lägg till text om själva köpupplevelsen
- **Läs andras omdömen** - Jämför olika butiker på ett enkelt sätt
- **Användarkonton** - Enkel registrering och inloggning
- **Butikshantering** - Grundläggande adminvy för att lägga till och uppdatera butiker

## Teknisk stack

- **Backend:** C# med ASP.NET Web API
- **Databas:** SQL Server
- **Frontend:** Angular med TypeScript
- **Hosting:** Microsoft Azure
- **Autentisering:** Grundläggande JWT-lösning

## Vad jag lärde mig

Det här var mitt första riktiga fullstackprojekt med Microsoft-teknik. Jag lärde mig att sätta upp ett REST-API, jobba med Entity Framework mot databasen och koppla ihop en Angular-frontend med en C#-backend. Att deploya till Azure gav mig också en bra första känsla för molnhosting i praktiken.
